# Alerts - Web App
> A very simple web app for sending out email alerts. It's pretty basic but I often have to make stuff like this and
> have struggled to find good examples online, so here's a public repo in case it's helpful to anyone else.
>


## Table of contents
* [Overview](#Overview)
* [How to Send Emails in Python](#how-to-send-emails-in-python)
* [Technologies](#technologies)

## Overview
You could use this to check for news articles, to see if a website of interest has changed, to look for a job you're 
interested in etc.

## How to Send Emails in Python
I did this an extremely simple way with a throw away gmail account for this purpose. The approach I took is detailed
quiet well [here](https://realpython.com/python-send-email/).

## Technologies
* Python 3.8
* Requests
* Pytest (lots of mocking!)
